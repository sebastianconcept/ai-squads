## Overview
The `sulaco` squad is a squad of agents collaborating intelligently to plan and execute tasks in assigned projects. The projects plan, mission, specs and tasks are created and found in `.squads-ai/projects/[PROJECT_NAME]` workspace. It serves as a smart context guiding system for all the agents in the squad feel oriented on what to do next.

## Core Purpose
- **Rust projects**: Efficiently plan and execute Rust projects
- **htmx projects**: Efficiently plan and execute web application projects using htmx
